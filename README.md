### Frontend
- React (Vite)
- Redux Toolkit
- TailwindCSS
- Axios
- Formik + Yup

- Node.js + Express
- TypeScript
- MongoDB + Mongoose
- JWT Authentication

- Create & Edit Monthly Budgets

- Delete Budgets & Categories
- Responsive UI (Mobile-first)
- Protected Routes
